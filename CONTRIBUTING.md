# Contributing

Obrigado por considerar contribuir com o projeto.

Por favor siga estas orientações para facilitar revisões e manter a qualidade:

- Fork o repositório e crie uma branch com nome descritivo: `feature/nome-curto` ou `fix/descricao`
- Faça commits pequenos e com mensagens claras (imperativo): `Adiciona validação de payload`.
- Abra Pull Requests contra a branch `main` e descreva o que muda e por quê.
- Inclua screenshots quando alterar a interface e testes ou passos para reproduzir bugs.

Revisões:
- PRs serão analisadas por mantenedores e podem receber pedidos de alteração.
- Siga os padrões de estilo do código: `dotnet format` para C#, `eslint` para frontend, `flake8`/`black` para Python.

Testes & CI:
- Antes de abrir PR, certifique-se de que o pipeline `CI` está verde localmente quando aplicável.

Discussões comerciais e suporte:
- Para propostas comerciais, licenciamento ou suporte, entre em contato via issues ou pelo contato listados no `README.md`.
